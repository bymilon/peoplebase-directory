# Problem & Solution: Context Windows

**Duration**: 30 seconds
**Campaign**: Problem/Solution
**Core Idea**: Traditional templates lack the documentation AI agents need to succeed.

## Scene 1 (0:00 - 0:05) - The Hook
- **Visual**: Screen recording of a developer watching an AI agent write a massive, broken React component full of inline styles. 
- **On-Screen Text**: Traditional templates break AI.
- **Narration**: Most dashboard templates break when you use AI coding agents. They use massive UI libraries with zero context.

## Scene 2 (0:05 - 0:15) - The Solution
- **Visual**: Panning across the `.wiki/` directory and `.sops/` folder in the PeopleBase repository.
- **On-Screen Text**: Spec-Driven Development in AI Studio.
- **Narration**: So we created PeopleBase Directory, fully built in Google AI Studio Apps Builder following spec-driven development pipelines. We feed the agent strict tokens and architectural records.

## Scene 3 (0:15 - 0:25) - The Outcome
- **Visual**: Developer dropping `DESIGN.md` and a specific SOP into Google AI Studio. The agent instantly outputs clean, responsive code.
- **On-Screen Text**: Clean code. Zero hallucinations.
- **Narration**: The result? Clean, production-ready code without the hallucinations. It just works.

## Scene 4 (0:25 - 0:30) - The CTA
- **Visual**: Clean PeopleBase dashboard on a desktop screen.
- **On-Screen Text**: Try the boilerplate today.
- **Narration**: Try the boilerplate today. Link in bio.
