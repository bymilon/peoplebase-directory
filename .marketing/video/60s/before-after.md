# Before & After: Design System Constraints

**Duration**: 60 seconds
**Campaign**: Before/After
**Core Idea**: Demonstrating the contrast between unconstrained AI generation and LLM-safe generation.

## Scene 1 (0:00 - 0:08) - The Hook (Before)
- **Visual**: Split screen. Left side: A messy React UI with inconsistent padding, a bright neon blue button, and overlapping text.
- **On-Screen Text**: Unconstrained AI.
- **Narration**: This is what happens when you let AI guess your design system. Random hex codes, broken flexbox, and inline styles everywhere.

## Scene 2 (0:08 - 0:20) - The Shift
- **Visual**: Focus shifts to the Right side: The clean, perfectly aligned PeopleBase dashboard.
- **On-Screen Text**: Built in AI Studio. Spec-Driven.
- **Narration**: Now look at this. This is PeopleBase Directory, fully built in Google AI Studio Apps Builder following spec-driven development pipelines to enforce an LLM-safe design.

## Scene 3 (0:20 - 0:40) - The Architecture
- **Visual**: Opening `DESIGN.md` to show the explicit color rules. Opening `src/index.css` to show the Tailwind `@theme` variables matching those rules.
- **On-Screen Text**: Markdown rules -> CSS Variables.
- **Narration**: Before we write any React, we define strict rules in a Markdown file, mapped directly to Tailwind CSS variables. When the AI generates a new chart or a sidebar, it is strictly forced to use our semantic tokens. No arbitrary values allowed.

## Scene 4 (0:40 - 0:50) - The Proof
- **Visual**: Fast cuts of generating three different components. Every single one matches the brand guidelines perfectly.
- **On-Screen Text**: Predictable outputs.
- **Narration**: The charts automatically adapt to dark mode. The navigation rail stays perfectly responsive. The code stays incredibly clean.

## Scene 5 (0:50 - 1:00) - The CTA
- **Visual**: The repository README page.
- **On-Screen Text**: Start using strict constraints.
- **Narration**: Stop fighting AI hallucinations and start using strict constraints. Grab the open-source repo today. Link in bio.
