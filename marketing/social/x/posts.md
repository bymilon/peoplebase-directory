## Feature
Magic hex codes break AI coding workflows. That's why PeopleBase Directory uses a strict DESIGN.md spec backed by native Tailwind CSS semantic variables. Predictable styling for humans and agents alike.

Repo: [LINK]

## Educational
Building for AI coding agents requires a new kind of architecture. We found that centralizing design rules in a simple markdown file, mapped to CSS variables, dramatically reduces generated UI inconsistencies. 

Learn how we did it in the PeopleBase wiki: [LINK]
