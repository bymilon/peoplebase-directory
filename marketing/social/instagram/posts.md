## Announcement
Say goodbye to bloated dashboard templates. PeopleBase Directory is now open-source. 

We built a lightweight, data-centric React SPA that prioritizes clean code over heavy UI libraries. Built with Vite and Tailwind CSS, it features an LLM-safe design system that keeps your styling consistent, whether you're coding by hand or using AI agents. 

Link in bio to check out the repository!

## Feature
Tired of AI agents hallucinating random colors in your UI? 

PeopleBase Directory introduces an LLM-safe design system. By strictly mapping a `DESIGN.md` specification to native Tailwind CSS variables, we eliminate magic hex codes. The result? Predictable, consistent styling across the entire application.

See how it works at the link in our bio.

## Educational
Your AI coding tools are only as good as the context you provide them. 

In PeopleBase Directory, we use the Open Knowledge Format (OKF) to build a Git-native second brain. Our `.wiki/` directory houses Architecture Decision Records, SOPs, and reusable prompts. Providing this structured context dramatically reduces hallucinations and improves output quality.

Learn more about OKF at the link in our bio.
