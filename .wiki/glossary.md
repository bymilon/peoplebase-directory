# Glossary

- **ADR**: Architecture Decision Record. A document that captures an important architectural decision made along with its context and consequences.
- **M3 Navigation Rail**: A Material Design 3 pattern that provides ergonomic navigation on desktop via a collapsed, icon-first rail that expands on interaction.
- **OKF**: Open Knowledge Format. A structured, Git-native, Markdown-based format for project documentation.
- **SPA**: Single Page Application.
- **Tailwind Semantic Tokens**: Named color variables in Tailwind CSS (like `slate`, `sky`) as opposed to raw hex codes.
